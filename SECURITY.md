# Security Policy

## Supported Versions

We only provide security updates for the `main` branch (latest version).

| Version | Supported |
| ------- | --------- |
| v0.x    | ✅ Yes    |
| < v0.x  | ❌ No     |

## Reporting a Vulnerability

We take security seriously at OCLKA. If you discover a security vulnerability, please send an email to **redflag@oclka.dev**.

Please include:

- A description of the vulnerability.
- Steps to reproduce it.
- Possible impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix. Please do not disclose the vulnerability publicly until it has been resolved.
